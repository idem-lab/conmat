# ABS lookup table of states, lga code and lga name

A dataset containing Australian Bureau of Statistics official short
state names, lga_code, and lga name.

## Usage

``` r
abs_lga_lookup
```

## Format

A data frame with 544 rows and 3 variables, arrange by state then LGA

- state:

  state - short state or territory name

- lga_code:

  official lga code

- lga:

  lga name
