# ABS state population data for 2020

Dataset containing Australian Bureau of Statistics state level
population data for 2020

## Usage

``` r
abs_state_age
```

## Format

A data frame with 168 rows and 3 variables:

- state:

  state - short state or territory name

- age_group:

  age group in five year bins from 0 to 99, then 100+

- population:

  population size

## Source

<https://www.abs.gov.au/statistics/people/population/national-state-and-territory-population/dec-2020#data-downloads-data-cubes>
