# Original school demographics for conmat

An internal dataset containing the original estimates of which fraction
of ages were attending school in Australia. These can be used inside of
[`fit_single_contact_model()`](https://idem-lab.github.io/conmat/reference/fit_single_contact_model.md)
and
[`fit_setting_contacts()`](https://idem-lab.github.io/conmat/reference/fit_setting_contacts.md).

## Usage

``` r
conmat_original_school_demographics
```

## Format

A data frame with 121 rows and 2 variables:

- age:

  0 to 120

- school_fraction:

  fraction of population at school

## Source

Census of Population and Housing, 2016, TableBuilder
