# Check dimensions

An internal function used within
[`apply_vaccination()`](https://idem-lab.github.io/conmat/reference/apply_vaccination.md)
to warn users of incompatible dimensions of data and the next generation
matrices

## Usage

``` r
check_dimensions(ngm, data)
```

## Arguments

- ngm:

  list with next generation matrices at different settings

- data:

  data frame
